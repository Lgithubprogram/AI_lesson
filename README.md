# AI_lesson Listen the story behind the picture
## 使用文档
环境配置见 requirements.txt
本项目开发时曾使用jupyter，所包含的库有部分没用上。
打开终端，进入你所配置的环境
```
conda activate your enviroment_name
```
进入文件夹.
```
cd your_AI_Lesson_absolute_path/Model
```

以streamlit运行app.py
```
streamlit run app.py
```
接下来就可以上传你的图片了
首次运行遇见这种情况直接回车即可
